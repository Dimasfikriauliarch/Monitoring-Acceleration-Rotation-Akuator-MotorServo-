# Monitoring-Acceleration-Rotation-Akuator-MotorServo-
Penggunaan accelerometer sensor bisa disematkan pada akuator/penggerak, sehingga bisa membuat monitoring maintenance pada Akuator/Penggerak (motor servo), ketika motor servo yang seharusnya bergerak ketika mendeteksi sensor suhu &lt;10 Celsius atau >10 Celsius, bisa dilihat dengan bergerak atau tidak nilai acceleration&amp;rotation yang telah disematkan.
